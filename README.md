# 🎵 Rhythm Flow – Music Player Web Application

A responsive and interactive **web-based music player** built using **HTML, CSS, and JavaScript** for hands-on front-end development practice.

---

## 📖 Overview

Rhythm Flow is a modern music player application designed to demonstrate practical knowledge of **UI design, JavaScript Audio API, and DOM manipulation**.  
It provides smooth audio controls, playlist interaction, and a visually appealing dark-themed interface.

---

## 🚀 Features

- ▶️ Play / Pause music
- ⏭️ Next & Previous track controls
- 🔊 Volume control with mute option
- 📊 Seek bar with real-time progress
- 🔍 Live song search functionality
- 📂 Playlist and popular songs section
- 🎨 Smooth animations with dark UI
- 📱 Fully responsive layout

---

## 🛠️ Tech Stack

- **HTML5** – Structure and semantics
- **CSS3** – Flexbox, animations, and styling
- **JavaScript (ES6)** – Audio API and DOM handling
- **Bootstrap Icons** – Icons
